PROCEDIMIENTO PARA LA VALIDACIÓN DE CALIFICACIÓN PLATAFORMA CANVAS, APLICACIÓN DE CONTROL GPR, SILABO

Descripción del Procedimiento

Este procedimiento establece los pasos y requisitos para la validación de calificaciones en la plataforma Canvas, aplicación de control GPR y el uso del sílabo, asegurando el cumplimiento de normativas académicas y estándares institucionales.
